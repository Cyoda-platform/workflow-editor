# `@cyoda/workflow-monaco`

Monaco integration for Cyoda workflow JSON editing, validation, markers,
and patch generation.

## Install

```sh
npm install @cyoda/workflow-core @cyoda/workflow-monaco monaco-editor
```

## Highlights

- Register the Cyoda workflow JSON schema with Monaco
- Convert validation issues into Monaco markers
- Attach a controller that emits `replaceSession` patches from JSON edits
