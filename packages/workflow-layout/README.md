# `@cyoda/workflow-layout`

ELK-based layout helpers for positioning and routing Cyoda workflow graphs.

## Install

```sh
npm install @cyoda/workflow-core @cyoda/workflow-graph @cyoda/workflow-layout
```

## Highlights

- Compute graph layouts with ELK
- Support preset layout options for common workflow views
- Produce node positions and edge routing data for consumers
